## Overview

The **Ecommerce Admin** project is an admin dashboard designed to manage an ecommerce website efficiently. It provides seamless state management, real-time notifications, authentication, and data visualization.

## Key Features

- **State Management**: Implemented using Redux and Redux Toolkit for a smooth user experience.
- **Notifications**: Integrated Toastify for real-time user alerts and system notifications.
- **Authentication & Authorization**: Secure login system using JSON Web Tokens (JWT) and bcrypt.
- **Data Visualization**: Charts and analytics powered by Ant Design and Graph components for better insights.

## Technologies Used

- **React**: Frontend framework for building an interactive UI.
- **Node.js & Express.js**: Backend for handling API requests.
- **Redux & Redux Toolkit**: Efficient global state management.
- **Toastify**: Enhanced user experience with real-time notifications.
- **Ant Design & Graph**: For creating dynamic and informative data visualizations.

## Outcome

The Ecommerce Admin panel streamlines ecommerce management by providing a powerful dashboard with enhanced security, real-time updates, and intuitive analytics. It is a scalable solution for handling online store operations efficiently.
