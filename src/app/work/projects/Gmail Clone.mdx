## Overview

[Gmail Clone](https://cerulean-banoffee-542bb7.netlify.app/) is a feature-rich email application that replicates the core functionalities of Gmail, including email sending and user authentication.

## Key Features

- **Email Sending**: Implemented using Nodemailer for reliable email delivery.
- **Authentication & Authorization**: Secured with JSON Web Tokens (JWT) and bcrypt.
- **UI Design**: Built with Material UI and Styled Components for a sleek interface.
- **Database**: Uses MongoDB for storing user data and emails.

## Technologies Used

- **Front-end**: React, Material UI, Styled Components
- **Back-end**: Node.js, Express.js, Nodemailer
- **Database**: MongoDB
- **Other Tools**: JWT, Bcrypt
