## Overview

[Ecommerce Website](https://spotify-eccomerce.netlify.app/) is a fully functional shopping platform that includes product browsing, cart management, authentication, and payment processing.

## Key Features

- **Form Validation**: Implemented using Formik and Yup.
- **Payment Gateway**: Integrated Razorpay for secure transactions.
- **Notifications**: Added Toastify for better user feedback.
- **State Management**: Used Redux and Redux Toolkit for seamless data flow.
- **Product Ratings**: Enabled rating functionality using React Star Components.
- **Image Hosting**: Utilized Cloudinary for product image storage.
- **Authentication & Security**: Implemented JWT-based authentication and password hashing using Bcrypt.

## Technologies Used

- **Front-end**: React, React Redux, Redux Toolkit
- **Back-end**: Node.js, Express.js
- **Database**: MongoDB
- **Others**: Cloudinary, Formik, Yup, Toastify, Razorpay, JWT, Bcrypt
