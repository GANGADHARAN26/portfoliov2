## Overview

[LMS Analytics Research](https://dashboard-analytics-lll.netlify.app/) is a static research demo page that explores the implementation of an analytics dashboard for an LMS system, providing insights into user interactions and performance metrics.

## Key Features

- **Data Visualization**: Implemented using React Charts for interactive graphs.
- **Material UI Design**: Utilized MUI components for a sleek and modern UI.
- **Dashboard Research**: Demonstrates a structured approach to analytics dashboard development.

## Technologies Used

- **Front-end**: React, MUI (Material UI), React Charts
