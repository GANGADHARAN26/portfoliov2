## Overview

[Netflix Clone](https://shiny-bavarois-bdc5af.netlify.app/) is a feature-rich streaming platform that replicates core functionalities of Netflix, including authentication, movie browsing, and a wishlist feature.

## Key Features

- **User Authentication**: Firebase authentication for secure login and user management.
- **Movie Data**: Integrated TMDB API for fetching and displaying movie details.
- **Wishlist Functionality**: Users can save movies to their personal wishlist.
- **UI Design**: Styled Components for a sleek and responsive user interface.

## Technologies Used

- **Front-end**: React, Styled Components
- **Back-end**: Node.js, Express.js
- **Database & Auth**: Firebase
- **Movie API**: TMDB API
